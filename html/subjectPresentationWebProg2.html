Web Programming II

ECTS: 7

Web Programming II provides advanced training in server-side web development, focusing on MVC architecture, database integration, and security. Students learn to implement MVC patterns, design RESTful APIs, and utilize Node.js for server-side scripting with databases like MySQL and MongoDB. The course covers ExpressJS for building robust web applications, emphasizes security measures, and teaches asynchronous client-server communication. By mastering these skills, students can develop dynamic and secure multiplatform web applications effectively.
