Mobile and Ubiquitous Computing

ECTS: 6.0

Mobile and Ubiquitous Computing focuses on developing skills in Web Mobile Applications using React and Hybrid Mobile Applications with React Native for Android and iOS. The course covers React essentials including components, props, state management, lifecycle methods, hooks, and best practices for Hybrid Mobile Development. It also includes core React Native concepts like UI design, interaction handling, data storage, sensors, and deployment strategies.
