Web Technologies

ECTS: 7

Web Technologies equips students with essential skills for developing modern web applications. The course covers the architecture and operation of the World Wide Web, key protocols and standards, and tools like HTML5, CSS, and JavaScript. Students learn to design responsive web pages, stylize them with CSS, and implement dynamic features using client-side scripting. Understanding these technologies is crucial for creating interactive and cross-platform web applications, preparing students for the demands of the web development industry.
